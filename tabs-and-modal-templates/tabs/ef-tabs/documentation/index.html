Effective Tabs Documentation

  • Created: 07/18/2018
  • By: ThemeSelection
  • Email: themeselection@gmail.com

Effective Tabs is a animated, responsive solution for creating beautiful tabbed navigation. It has different layouts, Animations, responsive design, 20 color schemes and can be easily integrated to any design.

Integration

  1. Download the zipped pack from ThemeSelection and extract to a folder on your computer.
  2. You need to include necessary css and js files to your project. 
    <link rel="stylesheet" href="css/demo.css">
<link rel="stylesheet" href="css/font-awesome.css">
<link rel="stylesheet" href="css/ef-tabs.css">
	
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script src="js/jquery.transit.js"></script>
<script src="js/ef-tabs.js"></script>
  3. Copy HTML code from any of the files.
  4. Now, You can replace demo content with yours.
  5. And Here you go :)

HTML structure

The HTML structure of the tabs consists of radios, checkbox, label and list.

Add below code to show radio button.

<div class="row">
	<div class="radio">
		<input type="radio" value="radio1" name="check" id="radio1" />
		<label for="radio1">Radio1</label>
	</div>
	<div class="radio">
		<input type="radio" value="radio2" name="check" id="radio2"/>
		<label for="radio2">Radio2</label>
	</div>
</div>

 

Add below code to show Checkbox.

<div class="row">
	<div class="checkbox">
		<input type="checkbox" name="agree" id="checkbox" />
		<label for="checkbox">Check box Text Here.</label>
	</div>
</div>

CSS structure

Effective Tabs includes 23 css files. 3 css Files are required and other 20 css files color scheme.

Required CSS Files.

  1. demo.css - Demo Styles.
  2. ef-tabs.css - Effective Tabs Main Stylesheet.
  3. font-awesome.min.css - To Display Icons.

The file "ef-tabs.css" contains all of the specific styling and separated into sections:

  • Font
  • Tabs CSS
  • Nav
  • Content
  • Fallback
  • grid system
  • typography
  • Icons
  • Individual tab styles

Color Schemes

If you would like to edit any section, find the appropriate label in the CSS file.

Tabs Styles

There are 14 different Tabs Styles. To use any of them, you need to add the appropriate class to tabs container. All available styles and classes listed in the table below.

Tabs Style Class Name
Bar ef-tabs-style-bar
Circle ef-tabs-style-circle
Circle Fill ef-tabs-style-circlefill
Fill UP ef-tabs-style-fillup
Flip ef-tabs-style-flip
Icon Box ef-tabs-style-iconbox
Icon Fall ef-tabs-style-iconfall
Line Box ef-tabs-style-linebox
Line Move ef-tabs-style-linemove
Line Triangle ef-tabs-style-linetriangle
Shape ef-tabs-style-shape
Top Line ef-tabs-style-topline
Tzoid ef-tabs-style-tzoid
Underline ef-tabs-style-underline

You need to add this class at outer wrapper of navigation and content area. Let's say, you want to add Bar tabs : 

<div class="ef-tabs ef-tabs-style-bar">
	<nav>
		<ul>
			<li></li>
			<li></li>
			<li></li>
		</ul>
	</nav>
	<div class="content-wrap">
		<section></section>
		<section></section>
		<section></section>
	</div>
</div>

Animations

Attribute Name Animation Details
fade_left_fade_right Fade Out from left Fade In from Right
fade_right_fade_left Fade Out from Right Fade In from Left
fade_slide_from_bottom Fade Out and Slide from bottom
fade_slide_from_left Fade Out and Slide from left
fade_slide_from_right Fade Out and Slide from right
fall Fall From Right-top
flip_bottom Flip From bottom
flip_left Flip From left
flip_right Flip From right
flip_top Flip From top
glue_left_slide_from_right Glue left and Slide to back from right
glue_right_slide_from_left Glue right and Slide to back from left
glue_top_slide_from_bottom Glue top and Slide to back from bottom
move_bottom_scale_up Move bottom Scale UP
move_left_scale_up Move left Scale UP
move_right_scale_up Move right Scale UP
newspaper Newspaper throw
push_bottom_pull_top Push Bottom Pull from top
push_left_pull_right Push left Pull from right
push_right_pull_left Push right Pull from left
push_top_pull_bottom Push top Pull from bottom
scale_down_scale_down Both scale down
scale_down_scale_up Scale down and Scale up
scale_down_slide_from_bottom Scale down and Slide from bottom
scale_down_slide_from_left Scale down and Slide from left
scale_down_slide_from_right Scale down and Slider from right
scale_down_slide_left-slide_left_scale_up Scale down-Slide left and Slide left-Scale up
scale_down_slide_right-slide_right_scale_up Scale down-Slide right and Slide right-Scale up
scale_up_scale_up Both scale up
side_left_side_right Move out from left side and move in from right
side_right_side_left Move out from right side and move in from left

		</ul>
	</nav>
	<div class="content-wrap">
		<section></section>
		<section></section>

	</div>
</div>

AJAX

You can set AJAX call also. Set AJAX call page url in data-content-url attribute. You have to add some code like below : 

<div id="ajax" data-content-url="content.html"></div>

Required* You have to set id="ajax".

I have this content.html file in same folder so I have not used full URL.

 

Map

We have used google map js to diaplay map. you need to add script for Map

<script src="https://maps.googleapis.com/maps/api/js"></script>

Now, Map in HTML. you have to add latitude and longitude in data-lat and data-lng respectively. You can also add information to display when you click on map pin. 

<div id="map" data-lat="23.023348" data-lng="72.575623"></div>
<div class="map-info">Map Info</div>

Required* You have to set id="map" and info class as class="map-info".

 

Image

To Include image you need to add simple img tag. That image will be displayed with full width.

 

Video

Here, you can add youtube and vimeo videos. To add this videos you need to add just iFrame of the video.

To add YouTube video, search/open video on YouTube. Find Share link and get it's Embed iFrame.

To add Vimeo video, search/open video on Vimeo. You will find Share link on right side of the video. When you click on that link it will popup a window which contains Embed iFrame.

 

iFrame

You can load whole site to the tab. For this add code below :

<div class="iframe">
	<iframe src="http://www.apple.com/" frameborder="0" scrolling="auto" height="1400" class="e-frame"></iframe>
</div>

Steps

You can set Previous Next steps using anchor tag. You have to use classes for previous and next links.

Required* You have to set class="next" for next link and class="previous" for previous link.

<div class="ef-tabs ef-tabs-style-bar"  data-effect="fade_left_fade_right">
	<nav>
		<ul>
			<li><a href="#sec-1">Home</a></li>
			<li><a href="#sec-2">About</a></li>
			<li><a href="#sec-3">Settings</a></li>
		</ul>
	</nav>
	<div class="content-wrap">
		<section id="sec-1"><a href="#" class="next"><span>Next step</span></a> →</p></section>
		<section id="sec-2">← <a href="#" class="previous">Previous step</a> | <a href="#" class="next">Next step</a> →</section>
		<section id="sec-3"><a href="#" class="previous">Next step</a> →</section>
	</div>
</div>

No need to set href value. Just set it to '#'.
